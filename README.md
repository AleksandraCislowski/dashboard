
# Northstar Commerce

Northstar Commerce is a portfolio project for an e-commerce operations
dashboard. The product is designed to help teams monitor revenue, orders,
conversion, channel efficiency, and customer health from a single workspace.

## Product Focus

- Daily performance monitoring for e-commerce operations teams
- Fast visibility into revenue trends, demand signals, and order risk
- A supporting analytics workspace for digging into operational records

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Starts the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run lint`

Runs ESLint for the Next.js app.

### `npm run build`

Creates an optimized production build.
